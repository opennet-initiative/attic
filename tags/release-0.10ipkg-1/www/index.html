Hallo!

Dies ist ein Opennet-WLAN-Access-Point, auf dem OpenWRT mit Opennet-Firmware Paket Version 0.10ipkg-1 "snow crash" läuft. Technische Details über diesen Access-Point sind auf der Statusseite nachzulesen. Informationen über Opennet finden sich im Internet unter http://opennet-initiative.de/.

Hinweis: Diese Software wurde als Paket auf OpenWRT installiert. Das Webfrontend und wichtige Konfigurationsdateien wurden vom Freifunk-Projekt übernommen und an Opennet angepasst. Zusammengehackt wurde die opennet-firmware von Rene, Sebastian und Daniel. Dank gilt auch allen, die durch ihre Tipps und Ratschläge diese Software verbessert haben. Fragen, Anregungen und Problemberichte - aber auch weitere ProgrammiererInnen - sind immer willkommen.

Hallo!

Dies ist ein Opennet-WLAN-Access-Point, auf dem die Opennet-Firmware Version 0.9 "rock mole" läuft. Technische Details über diesen Access-Point sind auf der Statusseite nachzulesen. Informationen über Opennet finden sich im Internet unter http://opennet-initiative.de/.

Hinweis: Diese Software basiert auf der ganz hervorragenden Linux-Distribution OpenWRT (release whiterussian rc2). Im Gegensatz zu anderen Firmwares ermöglicht OpenWRT das zusätzliche Installieren von Software-Paketen ohne dass auf erweiterte Versionen einer Firmware gewartet werden muss. Das Webfrontend und wichtige Konfigurationsdateien wurden vom Freifunk-Projekt übernommen, dem auch für vorhergehenden Versionen der Dank gilt. Mehr dazu im Internet unter http://www.freifunk.net/.

Zusammengehackt wurde die opennet-firmware von Rene und Sebastian (opennet-firmware AT absorb DOT it).
